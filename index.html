🏢 営業代理店管理システム

システム稼働状況

バックエンド: http://localhost:3001
フロントエンド: http://localhost:3000 (React開発サーバー)

主要機能一覧

📊 ダッシュボード

売上状況や代理店実績を一覧表示。組織全体の売上サマリーも確認可能。

🏢 代理店管理

代理店の登録・編集・削除。階層構造の管理とTier別制限機能付き。

💰 売上管理

売上データの登録と管理。編集・削除機能を含む完全な売上管理システム。

💵 報酬管理

階層報酬の自動計算と管理。源泉徴収税の自動計算機能付き。

📄 請求書管理

請求書・領収書のPDF生成とダウンロード。支払明細書の発行も可能。

📦 商品管理

商品情報の登録と管理。報酬率の設定も可能。

実装済み機能

  • JWT認証による安全なログイン機能
  • 階層型代理店構造（最大4階層）
  • Tier別子代理店数制限（Tier1:100社、Tier2:50社、Tier3:30社）
  • 自動報酬計算システム（基本報酬 + 階層ボーナス）
  • 源泉徴収税の自動計算（報酬額の10.21%）
  • 月次バッチ処理（自動締め処理・報酬計算）
  • Cronスケジューラーによる定期タスク実行
  • PDF生成機能（請求書・領収書・支払明細書）
  • 組織全体の売上サマリー表示
  • データ整合性チェック機能

バッチ処理コマンド

  • npm run batch:monthly - 月次締め処理
  • npm run batch:commission - 報酬計算実行
  • npm run batch:backup - 日次バックアップ
  • npm run batch:reminder - 支払い期日通知
  • npm run batch:check - データ整合性チェック
  • npm run scheduler - Cronスケジューラー起動
